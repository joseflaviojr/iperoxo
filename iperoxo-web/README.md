# Ipê-roxo

Exemplo de aplicação Web.

## Versão Atual

1.0-A14

Padrão de versionamento: [JFV](http://joseflavio.com/jfv)

## Requisitos

* Git >= 1.8
* Java >= 1.8
* Gradle >= 2.0
* Apache TomEE Plus >= 1.7.3

## Desenvolvimento

Configuração do projeto para Eclipse ou IntelliJ IDEA.

    gradle eclipse
    gradle cleanIdea idea

## Compilação

    gradle clean build
